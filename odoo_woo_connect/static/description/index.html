USER GUIDE

WooCommerce and Odoo Configuration

Step 1 : Go to Settings/ Sale and Select Delivery Method and Customer Address

Step 2 : Go to Settings/ Users & Companies/Users

Step 3 : Reload the page once done with the above configuration settings and you will see "Odoo WooCommerce Connector" in the menu

Step 4 : Configure your Instance by providing Location(URL), Consumer key, Consumer secret key, Version, Username and Password

Step 5 : After filling up the details, to check if entered credentials are correct, click on the "Check Connection" button

Step 5 : On successful Test connection, a popup with success message gets displayed.
